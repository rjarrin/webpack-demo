# webpack-demo

Demo following webpack use

# Basic Setup (getting-started)

- npm init -y
- npm install webpack webpack-cli --save-dev
- Create new project structure
  > webpack-demo
  > |- package.json
  > |- package-lock.json
  > |- index.html
  > |- /src
  > |- index.js
